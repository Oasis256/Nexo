# Skeleton Module for NexoPOS

A comprehensive demonstration module showcasing **all major features** of NexoPOS.

## Features Demonstrated

- Complete CRUD system
- Database migrations
- Routes (web + API)
- Controllers with DI
- Eloquent models
- Service layer
- Permission system
- Menu integration
- Blade templates
- Settings pages
- Event system
