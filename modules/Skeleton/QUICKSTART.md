# Quick Start - Skeleton Module

## Installation Complete!

The Skeleton module is ready to use.

## Access URLs

1. Dashboard: http://nexo.nlp/dashboard/skeleton
2. Manage Items: http://nexo.nlp/dashboard/skeleton/items
3. Add Item: http://nexo.nlp/dashboard/skeleton/items/create
4. Features: http://nexo.nlp/dashboard/skeleton/features
5. Settings: http://nexo.nlp/dashboard/skeleton/settings

## First Steps

1. Login to NexoPOS admin
2. Look for "Skeleton" menu in sidebar (cube icon)
3. Click "Dashboard" to start exploring
